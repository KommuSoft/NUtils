IMultiList4<T> An interface specifying a list of lists of lists of lists.
IQueue<TElement> An interface specifying a queue. A queue is a collection that works First-In-First-Out (FIFO).
IUniqueQueue<TElement> An interface describing an object that works as a FIFO-queue but where each element can only occur once.

NondeterministicFiniteAutomaton<TStateTag,TEdgeTag,TCollection> An implementation of the INondeterministicFiniteAutomaton`2 interface that uses a number of nodes that are linked together.
