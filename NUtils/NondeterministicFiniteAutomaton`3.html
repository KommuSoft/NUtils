NondeterministicFiniteAutomaton<TStateTag,TEdgeTag,TCollection> Generic Class

An implementation of the INondeterministicFiniteAutomaton`2 interface that uses a number of nodes that are linked together.

NondeterministicFiniteAutomaton Constructor

Initializes a new instance of the NondeterministicFiniteAutomaton`3 class based on tags with a given list of edges, the tag of the initial state (initialStateTag) and a list of accepting states (acceptingStateTags).

Syntax

public NondeterministicFiniteAutomaton (IEnumerable<Tuple<TStateTag, TEdgeTag, TStateTag>> edges, TStateTag initialStateTag, IEnumerable<TStateTag> acceptingStateTags)

Parameters

edges
The list of Tuple`3 instances representing edges to be added to the new NondeterministicFiniteAutomaton`3.
initialStateTag
The tag of the state that is the initial state.
acceptingStateTags
The list of tags representing the accepting states.

Remarks

Edges with tags that are not associated with any state will result in the creation of new IState`2 instances in this NondeterministicFiniteAutomaton`3.

If the given initialStateTag does not correspond with any registered state, a IState`2 instance will be created.

For each tag in the list of acceptingStateTags, the "first" IState`2 that matches the tag will be picked, if no IState`2 instance matches the given tag, the tag is ignored.

RegisterEdge Method

Register a state for the given sateTag. The new state doesn't contain any edges.

Syntax

public NUtils.Automata.IEdge<TStateTag, TEdgeTag> RegisterEdge (TStateTag fromStateTag, TEdgeTag edgeTag, TStateTag toStateTag)

Parameters

fromStateTag
The tag of the state from which the edge originates.
edgeTag
The tag of the edge that is registered.
toStateTag
The tag of the state to which the edge maps.

Returns

A IEdge`2 instance that is either an already registered state with the given stateTag or a new state created.

Remarks

If there is no state associated with the fromStateTag or toStateTag, states are registered (using the . method) for these tags.

If there already exists an edge between the two given states with the given tag, no additional edge is registered.

This operation is not completely specific to the given automaton: if the state is shared with another automaton, the edge will be added to all the automata.
