NondeterministicFiniteAutomaton Constructor

Initializes a new instance of the NondeterministicFiniteAutomaton`2 class based on tags with a given list of edges, the tag of the initial state (initialStateTag) and a list of accepting states (acceptingStateTags).

Syntax

public NondeterministicFiniteAutomaton (IEnumerable<Tuple<TStateTag, TEdgeTag, TStateTag>> edges, TStateTag initialStateTag, IEnumerable<TStateTag> acceptingStateTags)

Parameters

edges
The list of Tuple`3 instances representing edges to be added to the new NondeterministicFiniteAutomaton`2.
initialStateTag
The tag of the state that is the initial state.
acceptingStateTags
The list of tags representing the accepting states.

Remarks

Edges with tags that are not associated with any state will result in the creation of new IState`2 instances in this NondeterministicFiniteAutomaton`2.

If the given initialStateTag does not correspond with any registered state, a IState`2 instance will be created.

For each tag in the list of acceptingStateTags, the "first" IState`2 that matches the tag will be picked, if no IState`2 instance matches the given tag, the tag is ignored.
