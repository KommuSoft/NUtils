DoubleBlanket<TEnv,TChild> Generic Method

Enumerate the closests descendants of root that match the the given enumerate predicate, but nodes are only expanded if the given expand predicate. Descendants are enumerated with an "environment": the closest descendant of type TEnv that satisfies the environment predicate.

Syntax

public static IEnumerable<Tuple<TEnv, TChild>> DoubleBlanket<TEnv, TChild> (this TChild root, Predicate<TChild> expand, Predicate<TChild> enumerate, Predicate<TEnv> environment)
where TEnv : class
where TChild : NUtils.Designpatterns.IComposition<TChild>

Type Parameters

TEnv
The type of the environment, used to determine the environment "inodes".
TChild
The type of the elements to be enumerated.

Parameters

root
The root from which the double blanket is calculated.
expand
A predicate that determines which node must be expanded.
enumerate
A predicate that determines which nodes must be enumerated.
environment
A predicate that determines whether an inode is selected as the closest environment.

Returns

A list of tuples of the environment instance and the associated descendant to be enumerated.

Remarks

The default value of TEnv is used for items where no environment is yet defined.

The root is tested first as a potential environment.

It is possible that an element is enumerated with an environment that is equal to itself.

Remarks

One is not considered to be a descendant from oneself. The given root is not enumerated.

Descendants are enumerated left-to-right and top-to-bottom: a child is generated before the enumeration of its children.

All items are enumerated at their first occurence.

This method provides additional speedup because already enumerated items are not expanded anymore. This method is thus more efficient than putting a unique constraint on the output of .
