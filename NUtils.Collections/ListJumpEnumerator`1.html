Jump Method

Move the given number of steps forward in to the enumerator process.

Syntax

public bool Jump (int steps)

Parameters

steps
The number of steps to move forward.

Returns

true if the jump was successful (if the cursor is standing on an element, or the number of steps is zero or less; otherwise false.

Exceptions

Type Reason
InvalidOperationException The collection was modified after the enumerator was created.

Remarks

If the number of jumps is zero or less, nothing happens and the answer is always true, the cursor does not return back.
