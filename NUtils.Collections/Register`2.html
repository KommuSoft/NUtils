Register<TKey,TValue> Generic Class

A basic implementation of a IRegister`2 where the collection is by default List`1. A register is a datastructure that stores items based on a TKey for fast lookups.

Syntax

public class Register<TKey, TValue> : Register<TKey, TValue, List<TValue>>

Type Parameters

TKey
The key on which the given items are stored.
TValue
The items to store in the register.
