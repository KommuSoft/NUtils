RangeEnumerable Method

Enumerates the integer values between the given lower bound (inclusive) and the given upper bound, exclusive, optionally with a given delta per step.

Syntax

public static IEnumerable<int> RangeEnumerable (int lower, int upper, int delta)

Parameters

lower
The given lower bound, inclusive.
upper
The given upper bound, exclusive.
delta
The given difference per step, optional, by default equal to one (1).

Returns

A IEnumerable`1 that enumerates the integer values between lower (inclusive) and upper (exclusive) with steps of delta.

Remarks

The given IEnumerable`1 is guaranteed to be finite if delta is not equal to zero (0), regardless of the value of lower, upper, this due to the overflow of int values.
