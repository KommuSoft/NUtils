BitUtils Class

A utility class that specifies usefull operations on bit strings, bit tiles, etc.

Syntax

public static class BitUtils

Remarks

A bit tile is defined as a 64 bit string that is ordered as a 8x8 matrix (left to right, top to down).

PackUlong Method

A coroutine that packs 64 booleans into a single ulong. In case the booleans are exhausted and the length is not modulo 64, the booleans are padded with false values.

Syntax

public static IEnumerable<ulong> PackUlong (IEnumerable<bool> data)

Parameters

data
A IEnumerable`1 instance of bool values that will be grouped together.

Returns

An IEnumerable`1 of ulong instances, such that each ulong represents the value of 64 bool values.

Remarks

The result is calculated lazily, thus IEnumerable`1 instances without an end are supported.
