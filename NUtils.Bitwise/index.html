Type Description
BitUtils A utility class that specifies usefull operations on bit strings, bit tiles, etc.
CompactBitVector An implementation of an NUtils.Bitwise.IBitVector, the implementation uses ulong data in an array fashion.
IBitMatrix An interface specifying a twodimensionally aranged array of bits. A bitmatrix does not have to be fully modifyiable: sometimes only the upper right corner of the matrix can be set, etc.
IBitVector A vector of bits. This should be implemented in a compact way.
IConflictMiner An interface specifying a miner for conflicts. Such miner has the ability to detect conficts between indices and can return the conflict classes.
INumberVector An interface specifying a vector of numbers with arbitrary bit length. This is used to store the numbers in a compact way and perform fast operations.
