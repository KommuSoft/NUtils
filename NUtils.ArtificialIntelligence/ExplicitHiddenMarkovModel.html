GenerateInfluenceMatrix Method

Parameters

fsm
The given finite state machine to generate the influence matrix from.
strongIndex
The initial index to generate an influence matrix from.

Returns

A 2d-matrix that contains the influence matrix of the strongly connected group.

Remarks

The given initial index must be part of a strongly connected group. If the index is not part, the algorithm will loop.
