AddDirectedEdge Method

Add an directed edge from a node with the given fromIdentifier to a node with the given toIdentifier.

Syntax

public static void AddDirectedEdge (this IDotTextWriter writer, string fromIdentifier, string toIdentifier, params IEdgeDotAttribute[] dotAttributes)

Parameters

writer
The writer to which the edge is added.
fromIdentifier
The identifier of the first node of the edge.
toIdentifier
The identifier of the second node of the edge.
dotAttributes
An array of IEdgeDotAttribute instances that alter the way the edge is displayed.

Remarks

If one of the identifiers (fromIdentifier or toIdentifier), the edge is not added.

If there are no nodes defined with the given identifier, additional nodes will be added to the graph, this is the behavior of GraphViz DOT graphs.

If the given list of attributes is not effective, no attributes are added to the node.

If the given writer is not effective, nothing happens.
