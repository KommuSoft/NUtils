NullPathNode<T> Generic Class

An implementation of the INullPathNode`1 interface: a IPathNode`1 that is the epsilon character of the alphabet: not a node, mainly used for epsilon-edges in a nondeterminstic finite automaton.

Syntax

public class NullPathNode<T> : NUtils.PathNodeBase<T>, NUtils.Abstract.IValidater<T>, INullPathNode<T>, IPath<T>, IPathNode<T>
where T : NUtils.Designpatterns.IComposition<T>

Type Parameters

T
The type of the nodes on which this IPathNode`1 operates.

Remarks

Every NullPathNode`1 instance is equivalent with every other INUllPathNode`1 instance, as long as T is equivalent.

A INullPathNode`1 returns false on every IPathNode`1.Validate call.

An INullPathNode`1 will not match any item in a given tree, the tree will simply not be evaluated.
