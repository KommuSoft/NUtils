INullPathNode<T> Generic Interface

An interface that specifies a INullPathNode`1 instance: a node that is the epsilon in a string: in other words no node. The node doesn't accept any node, but is useful in an nondeterministic finite automaton.

Syntax

public interface INullPathNode<T> : NUtils.Abstract.IValidater<T>, IPath<T>, IPathNode<T>
where T : NUtils.Designpatterns.IComposition<T>

Type Parameters

T
The type of the nodes on which this IPathNode`1 operates.

Remarks

A INullPathNode`1 returns false on every IPathNode`1.Validate call.

An INullPathNode`1 will not match any item in a given tree, the tree will simply not be evaluated.
