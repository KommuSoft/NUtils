IsValidDistribution Method

Determines if the given list of probabilities is a valid distribution given the epsilon value.

Syntax

public static bool IsValidDistribution (this IEnumerable<double> probabilities)

Parameters

probabilities
The given list of probabilistic values to check.

Returns

true if all probabilistic values are larger than or equal to zero and sum up to one (given a certain tolerance); otherwise, false.

Remarks

If the given probabilities is not effective, the condition returns false.

The epsilon value is set on 1e-6d.

PickKUniform<T> Generic Method

Pick k unique uniformly items of of the given collection using an algorithm that runs linear with the size of the collection. This algorithm is only useful if k is proportional to the size of the collection.

Syntax

public static IEnumerable<T> PickKUniform<T> (this ICollection<T> collection, int k)

Type Parameters

T
The type of elements in the collection that will be enumerated.

Parameters

collection
The collection that contains the item to select.
k
The number of items to enumerate, should be larger than zero.

Returns

A IEnumerable`1 that enumerates k unique elements in the same order as how the items are enumerated from the collection. The probability of an item getting selected is uniform.

Exceptions

Type Reason
ArgumentNullException If the given collection is not effective.
ArgumentException If the collection generates a non-effective IEnumerator`1.

Remarks

If k is less than or equal to zero, no items are enumerated.

If the collection contains an item twice, it can be enumerated twice. The algorithm only guarantees that an item at the same index will not be enumerated twice.
