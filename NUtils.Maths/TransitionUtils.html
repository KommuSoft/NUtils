GetStronglyConnectedGroups Method

Enumerate the sets of strongly connected groups: indices that form a cycle.

Syntax

public static IEnumerable<IEnumerable<int>> GetStronglyConnectedGroups (this ITransition transition)

Parameters

transition
Documentation for this section has not yet been entered.

Returns

A IEnumerable`1 that contains the indices of strongly connected groups.

Remarks

The strongly connected groups are generated lazily: on demand the next group is generated.

There is no guarantee on the order of the generated groups, nor on the first element of every group.

Singleton groups are generated as well: indices that have a transition to themselves.

The algorithm is a special case of Tarjans algorithm especially optimized for the transition functions.
