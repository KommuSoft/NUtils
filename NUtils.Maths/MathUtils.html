Binomial Method

Calculate the binomium for the given nand k, or the number of ways you can pick k elements out of a collection of n.

Syntax

public static ulong Binomial (int n, int k)

Parameters

n
The number of available items.
k
The number of items to pick.

Returns

The number of ways to pick k items out of a collection of n items.

Remarks

This function will only work for small number due to ulong overflow.

If n is less or equal to k or equal to zero, one is returned.

If k is less than or equal to zero, one is returned.

If k is one, the result is always n, this is a special property of the binomial operator.

The algorithm is implemented to reduce the cases where overflow is reached, as a rule of the thumb: if the result can be be represented with 60 bits, it will probably work.

LogFactorialDivStirling Method

Approximate the natural logarithm of the factorial of n over the factorial of n minus k using the Stirling approximation.

Syntax

public static double LogFactorialDivStirling (int n, int k)

Parameters

n
The value of the factorial numerator.
k
The value to substract from n for the factorial divider.

Returns

An approximation for the natural logarithm of the factorial for the devision between the factorials of n and n-k.

Remarks

If n is less than or equal to zero (0), the result is double.NaN.

If k is negative, the approximate still works and in that case the divider is greater than the numerator.

If k is greater than or equal to n, the result is double.NaN.

The Stirling approximation is less accurate than the . approximation, but less computationally expensive as well.
