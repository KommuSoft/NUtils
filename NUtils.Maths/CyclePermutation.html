CyclePermutation Class

A permutation described as a sequence of integers that describe a cycle.

Syntax

public class CyclePermutation : NUtils.Abstract.INormalize, NUtils.Abstract.IValidateable

Remarks

For instance if the cycle is 2->5->7->3, then 2 is mapped on 5, 3 on 2, 5 on 7 and 7 on 3, the other indices map on themselves.

NUtils.Maths.CyclePermutation instances are permutations that are used if most of the elements map on themselves. It is not possible to represent all possible permutations with a NUtils.Maths.CyclePermutation instance.

An empty cycle represents a permutation that maps all indices to themselves.
