RegisterEdges<TStateTag,TEdgeTag> Generic Method

Register the list of edges with a list of tuples containing the tags of the initial and final state as well as the tag for the edge to create.

Syntax

public static void RegisterEdges<TStateTag, TEdgeTag> (this INondeterministicFiniteAutomaton<TStateTag, TEdgeTag> nfa, IEnumerable<Tuple<TStateTag, TEdgeTag, TStateTag>> edges)

Type Parameters

TStateTag
Documentation for this section has not yet been entered.
TEdgeTag
Documentation for this section has not yet been entered.

Parameters

nfa
The finite state automaton to which the edges must be added.
edges
A IEnumerable`1 of Tuple`3 instance containing the tag of the initial state, the tag of the edge to create and the tag of the final state.

Remarks

If there is no state associated with the fromStateTag or toStateTag, states are registered (using the . method) for these tags.

If there already exists an edge between the two given states with the given tag, no additional edge is registered.

If the given list of edges or the nfa is not effective, nothing happens.

This operation is not completely specific to the given automaton: if the state is shared with another automaton, the edge will be added to all the automata.
