INondeterministicFiniteAutomaton<TStateTag,TEdgeTag> Generic Interface

An interface representing a nondeterministic finite automaton.

Syntax

public interface INondeterministicFiniteAutomaton<TStateTag, TEdgeTag> : NUtils.Abstract.ICloneable<INondeterministicFiniteAutomaton<TStateTag, TEdgeTag>>, ICloneable

Type Parameters

TStateTag
The type of the tags that are assigned to the nodes.
TEdgeTag
The type of the tags that are assigned to the edges.

Remarks

Most implementations of this interface will require that the tags are unique per state: two states can't share the same tag.

RegisterEdge Method

Register a state for the given sateTag. The new state doesn't contain any edges.

Syntax

public IEdge<TStateTag, TEdgeTag> RegisterEdge (TStateTag fromStateTag, TEdgeTag edgeTag, TStateTag toStateTag)

Parameters

fromStateTag
The tag of the state from which the edge originates.
edgeTag
The tag of the edge that is registered.
toStateTag
The tag of the state to which the edge maps.

Returns

A IEdge`2 instance that is either an already registered state with the given stateTag or a new state created.

Remarks

If there is no state associated with the fromStateTag or toStateTag, states are registered (using the . method) for these tags.

If there already exists an edge between the two given states with the given tag, no additional edge is registered.

This operation is not completely specific to the given automaton: if the state is shared with another automaton, the edge will be added to all the automata.
