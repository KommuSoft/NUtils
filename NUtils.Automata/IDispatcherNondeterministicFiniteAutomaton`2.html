IDispatcherNondeterministicFiniteAutomaton<TStateTag,TEdgeTag> Generic Interface

An interface representing an INondeterministicFiniteAutomaton`2 with an embedded dispatcher to allocate new state tags.

Syntax

public interface IDispatcherNondeterministicFiniteAutomaton<TStateTag, TEdgeTag> : NUtils.Abstract.ICloneable<IDispatcherNondeterministicFiniteAutomaton<TStateTag, TEdgeTag>>, NUtils.Abstract.ICloneable<INondeterministicFiniteAutomaton<TStateTag, TEdgeTag>>, INondeterministicFiniteAutomaton<TStateTag, TEdgeTag>, ICloneable

Type Parameters

TStateTag
The type of the tags that are assigned to the nodes.
TEdgeTag
The type of the tags that are assigned to the edges.

Remarks

Most implementations of this interface will require that the tags are unique per state: two states can't share the same tag.

One can still define statetags oneself, but must be cautious when the dispatcher generates an already allocated tag.
